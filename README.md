### What's Gauntlet?

Gauntlet is a Game Trainer for the Assault Cube.

### Some Features

- [x] Ammo/Life/Vest increase;
- [x] ESP;
- [x] Aimbot;
- [x] Ghost Mode;

### Installation
Install the Python requirements, open the AC and run the main.py.

### Configuration
Resolution: 1366x768

Fullscreen: No
> Note: if you change your team you'll need to restart the Gauntlet (if you want to use the aimbot)
